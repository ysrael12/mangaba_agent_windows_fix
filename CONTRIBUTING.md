# Contribuindo com o Mangaba Agent

Obrigado pelo interesse em contribuir! 🥭

## Como contribuir

1. Faça um fork do repositório
2. Crie uma branch: `git checkout -b feat/minha-feature`
3. Faça suas alterações e commit: `git commit -m "feat: minha feature"`
4. Envie para o fork: `git push origin feat/minha-feature`
5. Abra um Pull Request

## Padrões

- Código em Python 3.10+
- Testes com pytest
- Mensagens de commit em português ou inglês

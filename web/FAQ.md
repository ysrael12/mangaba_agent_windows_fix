# ❓ Perguntas Frequentes (FAQ)

**Respostas rápidas para dúvidas comuns**

---

## 🚀 Começar

### P: Como faço login?
**R:** Abra o dashboard e use suas credenciais (email e senha) que foram fornecidas. Se não tem uma conta, peça ao administrador.

### P: Esqueci minha senha, o que faço?
**R:** 
1. Na tela de login, clique "Esqueci a senha"
2. Digite seu email
3. Verifique seu email (procure pasta SPAM)
4. Clique no link de reset
5. Crie nova senha

### P: Como mudo meu idioma?
**R:** 
1. Vá para ⚙️ **Configurações**
2. Procure "Idioma"
3. Escolha na dropdown
4. Página recarrega em novo idioma

### P: Posso usar em dispositivo mobile?
**R:** **Sim!** O dashboard é totalmente responsivo. Funciona em smartphone, tablet, laptop e desktop.

## 💬 Chat & Conversas

### P: Por que o agente não responde?
**R:** Verifique:
1. ✓ Está conectado à internet?
2. ✓ Servidor está online? (verifique "Agentes Ativos")
3. ✓ Agente está 🟢 ATIVO (não 🔴)?
4. ✓ Tem permissão de usar este agente?

Se ainda não funcionar, contate suporte.

### P: Quanto tempo leva para agente responder?
**R:** Geralmente **2-10 segundos**, dependendo:
- Complexidade da pergunta
- Tamanho da resposta
- Carga do servidor
- Velocidade da internet

Se demorar muito (>30s), pode ser timeout. Tente novamente.

### P: Meu histórico de chat vai ser perdido?
**R:** **Não!** Todas as conversas são salvas em "Minhas Sessões". Pode ver histórico completo a qualquer momento.

### P: Posso deletar uma conversa?
**R:** Sim! Em "Minhas Sessões":
1. Abra conversa
2. Clique [⋮] (menu)
3. Escolha "Deletar"
4. Confirme

### P: Agente comete erros nas respostas?
**R:** Sim, é normal! IAs não são 100% precisas. **Sempre verifique** informações críticas. Se problema recorrente, reporte ao admin.

### P: Posso exportar conversa?
**R:** Sim! Na conversa aberta:
1. Clique [⋮] (menu)
2. Escolha "Exportar"
3. Escolha formato (PDF, TXT, JSON)
4. Download automático

## 🤖 Agentes

### P: Como crio um novo agente?
**R:** 
1. Clique em "Criar agente" (menu Agentes)
2. Preencha: Nome, Descrição, Modelo
3. Clique [Criar]
4. Aguarde confirmação

### P: Qual modelo de IA escolho?
**R:** Depende do caso:
- **GPT-4:** Melhor qualidade, mais lento, mais caro
- **Claude:** Equilibrado, rápido
- **GPT-3.5:** Mais barato, respostas básicas

Dúvida? Use Claude como padrão.

### P: Posso editar agente depois de criar?
**R:** **Sim!** Em "Agentes Ativos":
1. Clique no agente
2. Vá para "Editar"
3. Modifique o que quiser
4. Clique [Salvar]

### P: Por que agente está em erro 🔴?
**R:** Causas comuns:
- Sem conexão com banco de dados
- Limite de requisições atingido
- API key expirada
- Servidor offline

Clique [Detalhes] para ver erro específico. Se não conseguir resolver, reporte.

### P: Como reinicio um agente?
**R:** Em "Agentes Ativos":
1. Clique no agente com problema
2. Clique [Reiniciar]
3. Aguarde 5-10 segundos
4. Status deve voltar a 🟢

### P: Posso deletar um agente?
**R:** **Cuidado, é permanente!**
1. Em "Agentes Ativos", clique no agente
2. Vá para "Editar"
3. Role para baixo, clique [Deletar Agente]
4. Confirme digitando nome do agente
5. Agente e histórico são deletados

## ⚙️ Configurações

### P: Como mudo meu nome no dashboard?
**R:** 
1. Vá para ⚙️ **Configurações**
2. Seção "Perfil"
3. Altere "Nome"
4. Clique [Salvar]

### P: Como mudo de tema?
**R:** 
1. Clique botão de tema 🌙☀️ no topo
2. Escolha: Claro, Escuro, Automático
3. Tema muda instantaneamente

**Ou em Configurações:**
1. ⚙️ **Configurações**
2. "Aparência" → "Tema"
3. Escolha opção
4. Salve

### P: Como ligo/desliga notificações?
**R:** 
1. ⚙️ **Configurações**
2. "Notificações"
3. Toggle On/Off
4. Escolha canais (Email, Push, etc)
5. Salve

### P: Como mudo password?
**R:** 
1. ⚙️ **Configurações**
2. "Segurança" → "Senha"
3. Clique [Alterar Senha]
4. Digite senha atual
5. Digite nova senha 2x
6. Clique [Atualizar]

### P: Como ativa 2FA (autenticação dupla)?
**R:** 
1. ⚙️ **Configurações**
2. "Segurança" → "2FA"
3. Clique [Ativar]
4. Escaneie QR code com app (Google Authenticator, Authy)
5. Digite código para confirmar

## 📊 Logs & Monitoramento

### P: Por que os logs estão vazios?
**R:** Pode ser:
- Nenhuma atividade ainda
- Filtros muito específicos
- Logs mais antigos (limpeza automática)
- Permissão limitada

Tente remover filtros e recarregar página.

### P: Como filtro os logs?
**R:** 
1. Vá para **Logs**
2. Use dropdowns: Tipo, Data, Status
3. Logs atualizam automaticamente
4. Clique [Limpar Filtros] para reset

### P: Quanto tempo os logs ficam salvos?
**R:** Geralmente **30-90 dias**, dependendo:
- Configuração do servidor
- Volume de atividade
- Plano de armazenamento

Logs antigos são deletados automaticamente.

### P: Como exporto os logs?
**R:** Se há botão [Exportar]:
1. Clique [Exportar]
2. Escolha formato: CSV, JSON, PDF
3. Download inicia automaticamente

## 🔐 Segurança & Privacidade

### P: Minhas conversas são privadas?
**R:** **Sim!** São criptografadas em trânsito (HTTPS). Apenas você e administrador podem ver.

### P: Dados são armazenados onde?
**R:** Geralmente em servidores da sua organização ou cloud seguro. Pergunte ao admin para detalhes específicos.

### P: Como me desconecto (logout)?
**R:** 
1. ⚙️ **Configurações**
2. Procure "Sair" ou "Logout"
3. Clique
4. Você volta para tela de login

**Automático:** Após 30 minutos de inatividade (pode variar).

### P: Alguém pode hackear minha conta?
**R:** Riscos são baixos se:
- ✓ Usa senha forte (8+ caracteres)
- ✓ Ativa 2FA
- ✓ Não compartilha senha
- ✓ Usa WiFi seguro (não público)

### P: Como deleto minha conta?
**R:** ⚠️ **PERMANENTE!**
1. ⚙️ **Configurações**
2. "Zona de Risco" → [Deletar Conta]
3. Digite sua senha para confirmar
4. Clique [Deletar Permanentemente]

Tudo será deletado em 24-48 horas.

## 🐛 Problemas Técnicos

### P: Dashboard é lento
**R:** Tente:
1. **Recarregue página:** Ctrl+R ou Cmd+R
2. **Limpe cache:** Ctrl+Shift+Delete
3. **Feche outras abas:** Libera memória
4. **Reinicie navegador:** Fecha completamente
5. **Tente outro navegador:** Chrome, Firefox, Safari
6. **Verifique internet:** Faça speedtest.net

Se ainda lento, pode ser servidor. Contate suporte.

### P: Página mostra erro 404
**R:** 
- Verifique URL (copie da barra de endereço)
- Use menu lateral em vez de digitar URL
- Recarregue página
- Se persistir, pode ser rota deletada

### P: Conexão recusada / Offline
**R:** 
1. Verifique internet (abra google.com)
2. Reinicie WiFi/rede
3. Verifique se servidor está online
4. Tente VPN se bloqueado regionalmente
5. Contate admin

### P: Agente responde com erro estranho
**R:** 
1. Copie mensagem de erro
2. Verifique **Logs** (pode ter mais detalhes)
3. Tente pergunta diferente
4. Se erro recorrente, reporte ao admin

### P: Não consigo fazer upload de arquivo
**R:** Verifique:
- Arquivo < limite de tamanho (geralmente 10MB)
- Formato suportado (PDF, TXT, CSV, etc)
- Permissão de upload (nem todo agente permite)
- Espaço em disco no servidor

Passo 1

Baixar e Instalar

É um comando só. Ele faz tudo automaticamente: baixa, instala e configura.

🐧 Linux, macOS, WSL2, Termux

Copie e cole este comando no seu terminal:

curl -fsSL https://raw.githubusercontent.com/
  dheiver2/mangaba-agent/main/scripts/install.sh | bash

O que esse comando faz:
1. Baixa o instalador
2. Instala Python 3.11
3. Instala Node.js
4. Instala todas as dependências
5. Configura tudo automaticamente

🪟 Windows (PowerShell)

Abra o PowerShell como administrador e cole:

iex (irm https://raw.githubusercontent.com/
  dheiver2/mangaba-agent/main/scripts/install.ps1)

Dica: Para melhor experiência, use WSL2 (Windows Subsystem for Linux). É como ter Linux dentro do Windows.

✅ Verifique se funcionou

Abra um novo terminal e digite:

mangaba --version

Se aparecer a versão, tudo certo! Se não, feche e abra o terminal novamente.

Passo 2

Configurar o Modelo

Agora vamos escolher qual "cérebro" usar. É como escolher o motor do carro.

🎯 Opção Rápida (Recomendada)

O OpenRouter tem um plano gratuito e acesso a 200+ modelos. É a melhor opção para começar.

# 1. Crie uma conta gratuita em openrouter.ai # 2. Gere uma chave de API # 3. Configure no Mangaba: mangaba model set openrouter/anthropic/claude-sonnet-4 # 4. Cole sua chave quando pedir

Por que OpenRouter?
• Plano gratuito disponível
• 200+ modelos para escolher
• Funciona em qualquer lugar
• Sem compromisso

🔧 Outras Opções

Se você já tem uma chave de outro provedor, pode usar:

# OpenAI (GPT-4o) mangaba model set openai/gpt-4o # HuggingFace (gratuito) mangaba model set huggingface/meta-llama/llama-3.3-70b # Ollama (roda no seu PC) mangaba model set ollama/llama3

Dica: Se não tem nenhuma chave, use o OpenRouter com o plano gratuito. É rápido e sem custo.

💡 Não sabe qual modelo escolher?

Para iniciantes: use openrouter/anthropic/claude-sonnet-4 (gratuito).
Para avançados: use openrouter/openai/gpt-4o (mais potente).
Para offline: use ollama/llama3 (roda no seu PC).

Se der problema

Não funciona? Veja aqui

Calma, tudo tem solução. Veja os problemas mais comuns:

❌ "Comando não encontrado"

Depois de instalar, você precisa fechar e abrir o terminal novamente.

# Ou recarregue o shell: source ~/.bashrc # ou source ~/.zshrc

❌ "API Key inválida"

Verifique se sua chave está configurada:

mangaba doctor # Mostra status de todas as chaves

🐌 Muito lento

Troque para um modelo mais rápido:

mangaba model set openrouter/meta-llama/llama-3.3-70b

Comece em minutos

Uma linha no terminal e você está pronto.

Passo 1

Instalação

Escolha seu sistema operacional e execute o comando.

🐧 Linux, macOS, WSL2, Termux

curl -fsSL https://raw.githubusercontent.com/
  dheiver2/mangaba-agent/main/scripts/install.sh | bash

Funciona em qualquer Unix-like. Instala uv, Python 3.11, Node.js e dependências.

🪟 Windows (PowerShell)

iex (irm https://raw.githubusercontent.com/
  dheiver2/mangaba-agent/main/scripts/install.ps1)

Beta. Inclui Git Bash portátil. Para melhor experiência, use WSL2.

Comandos

Referência Rápida

Os comandos mais usados do Mangaba Agent.

Comando Descrição
mangaba Iniciar conversa interativa
mangaba model Escolher provedor e modelo de LLM
mangaba tools Configurar quais ferramentas estão habilitadas
mangaba config set Definir valores individuais de configuração
mangaba gateway Iniciar o gateway de mensagens
mangaba setup Executar assistente de configuração completo
mangaba update Atualizar para versão mais recente
mangaba doctor Diagnosticar problemas na instalação
mangaba cron list Listar automações agendadas
mangaba logs Visualizar logs do agente
Avançado

Configuração Completa

Personalize o agente para seu caso de uso.

📄 Arquivo de Configuração

Edite ~/.mangaba/config.yaml para configurações avançadas.

# ~/.mangaba/config.yaml model: openrouter/anthropic/claude-sonnet-4 theme: dark language: pt-BR gateway: telegram: token: SEU_TOKEN terminal: backend: docker timeout: 180

🔑 Variáveis de Ambiente

Chaves de API ficam em ~/.mangaba/.env.

# ~/.mangaba/.env OPENROUTER_API_KEY=sk-or-... HF_TOKEN=hf_... EXA_API_KEY=... FAL_KEY=...

Nunca commite este arquivo no Git!

Solução de Problemas

Troubleshooting

Problemas comuns e como resolver.

🔧 Comando não encontrado

Após a instalação, recarregue o shell:

source ~/.bashrc # ou abra um novo terminal

❌ Erro de API Key

Verifique se a chave está configurada:

mangaba doctor # Mostra status de todas as chaves

🐌 Lentidão

Troque para um modelo mais rápido:

mangaba model set openrouter/meta-llama/llama-3.3-70b

Recursos

Tudo que você precisa em um agente

Do terminal ao Telegram, de scripts a automações — um único agente para tudo.

🧠
Loop de Aprendizado
Cria habilidades automaticamente após tarefas complexas. As melhora durante o uso. Busca FTS5 de sessões com sumarização por LLM.
💬
Multi-Plataforma
Telegram, Discord, Slack, WhatsApp, Signal e CLI — tudo de um único processo gateway. Continuidade entre plataformas.
🔧
40+ Ferramentas
Código, web, arquivos, APIs, banco de dados, mídia, comunicação. Tudo integrado e pronto para usar.
⏰
Automações Agendadas
Cron integrado com entrega em qualquer plataforma. Relatórios diários, backups noturnos — em linguagem natural.
🤖
Sub-Agentes
Spawn de sub-agentes isolados para fluxos paralelos. Colapsando pipelines multi-etapa em turnos de custo zero.
🌐
Roda em Qualquer Lugar
Local, Docker, SSH, Modal, Daytona, Vercel Sandbox, Singularity. Serverless com persistência.
Como funciona

Simples de instalar, poderoso para usar

Uma linha para instalar. Um comando para começar. O agente faz o resto.

01
Instale com uma linha
Execute o script de instalação no seu terminal. Funciona em Linux, macOS, WSL2 e Termux.
02
Configure seu modelo
Escolha entre OpenRouter, OpenAI, NVIDIA, Hugging Face ou seu próprio endpoint. Troque a qualquer momento.
03
Comece a conversar
O agente aprende com cada interação, cria habilidades automaticamente e evolui com você.
Arquitetura

Construído para escalar

Modular, extensível e pronto para produção.

Núcleo

AIAgent

O loop principal de conversação com orquestração de ferramentas, compressão de contexto e tracking de orçamento.

Ferramentas

Registry + Toolsets

Sistema de descoberta automática com 40+ ferramentas organizadas em toolsets por plataforma.

Gateway

Multi-Plataforma

Plugins

Sistema Extensível

Memória, provedores de modelo, engines de contexto e mais — tudo via plugins descobertos automaticamente.

Comece agora

Instalação rápida

Uma linha no terminal e você está pronto.

Linux, macOS, WSL2, Termux

curl -fsSL https://raw.githubusercontent.com/
  dheiver2/mangaba-agent/main/scripts/install.sh | bash

Windows (PowerShell)

iex (irm https://raw.githubusercontent.com/
  dheiver2/mangaba-agent/main/scripts/install.ps1)

Após a instalação

source ~/.bashrc
mangaba          # começar a conversar!
mangaba model   # escolher modelo
mangaba tools   # configurar ferramentas
mangaba gateway # iniciar Telegram, Discord, etc.

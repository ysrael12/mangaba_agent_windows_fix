Recursos

Tudo que você precisa em um agente

Mangaba Agent combina aprendizado contínuo, memória persistente e infraestrutura flexível.

🧠
Auto-aperfeiçoamento
Cria habilidades a partir de experiências e as refina automaticamente. Cada sessão o torna mais inteligente.
💾
Memória persistente
Recall entre sessões com busca FTS5, sumarização por LLM e modelagem de usuário. Nunca perde contexto.
🌐
Multi-plataforma
Terminal, Telegram, Discord, Slack, WhatsApp, Signal e Email — tudo de um único processo gateway.
🔌
200+ modelos
OpenRouter, OpenAI, HuggingFace, NVIDIA NIM e mais. Troque com um comando, sem lock-in.
⚡
Infraestrutura flexível
VPS de $5, cluster GPU ou serverless (Modal, Vercel Sandbox) com hibernação automática quando ocioso.
🤖
Sub-agentes paralelos
Spawne agentes isolados para fluxos paralelos. Divida trabalho complexo em agentes especializados.
⏰
Automações cron
Agendador integrado. Resumo diário de emails, relatórios semanais, monitoramento contínuo — sem supervisão.
🛠️
40+ ferramentas
Código, web, arquivos, APIs, bancos de dados, mídia e comunicação. Tudo pronto para usar.
🔒
Privacidade total
Rode 100% local. Seus dados ficam na sua máquina. Sem telemetria, sem envio externo.
Como funciona

O loop de
auto-aprendizado

Cada tarefa vira uma oportunidade de aprendizado. O agente extrai padrões, cria habilidades reutilizáveis e as melhora continuamente.

1

Você envia uma tarefa

Via terminal, Telegram, Discord ou qualquer plataforma conectada.

2

Mangaba executa com ferramentas

Usa 40+ ferramentas integradas para completar a tarefa de forma autônoma.

3

Habilidade é extraída

O loop de aprendizado analisa a solução e cria uma habilidade reutilizável.

4

Próxima sessão: mais inteligente

A habilidade é aplicada automaticamente em tarefas similares futuras.

Instalação

Pronto em minutos

Escolha sua plataforma e instale com um comando.

bashinstala e configura automaticamente
# Linux, macOS ou WSL2
curl -fsSL https://raw.githubusercontent.com/dheiver2/mangaba-agent/main/scripts/install.sh | bash
powershellinclui Python, Node.js e Git Bash
# PowerShell (executar como administrador)
iex (irm https://raw.githubusercontent.com/dheiver2/mangaba-agent/main/scripts/install.ps1)
bashinstalação manual
git clone https://github.com/dheiver2/mangaba-agent.git
cd mangaba-agent
pip install -e .
mangaba setup

bashprimeiros passos
# Configurar modelo
mangaba model set openrouter/anthropic/claude-sonnet-4-6

# Iniciar o agente
mangaba

# Ou conectar no Telegram
mangaba gateway telegram --token SEU_TOKEN

Instalar agora Ver recursos

Rode em VPS de $5, cluster GPU, ou infraestrutura serverless.

# Instalação em uma linha
curl -fsSL https://raw.githubusercontent.com/
  dheiver2/mangaba-agent/main/scripts/install.sh | bash

# Iniciar conversa
mangaba

# Trocar de modelo
mangaba model set openrouter/claude-sonnet-4

# Agendar automação
mangaba cron add "0 9 * * *" "resumo dos emails"

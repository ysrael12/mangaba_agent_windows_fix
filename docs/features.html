Automações

Agende tarefas em linguagem natural

Cron integrado com entrega em qualquer plataforma.

⏰ Agendador Cron

Crie automações complexas com sintaxe familiar ou linguagem natural.

# Resumo diário de emails às 9h mangaba cron add "0 9 * * *" "resumo dos emails" # Backup semanal às sextas mangaba cron add "0 18 * * 5" "faça backup" # Monitoramento a cada hora mangaba cron add "0 * * * *" "verifique alertas" # Listar automações mangaba cron list

🤖 Sub-Agentes

Delegue tarefas paralelas para sub-agentes isolados.

# Spawne agentes para tarefas paralelas agente_pesquisa = mangaba.spawn("pesquise sobre X") agente_codigo = mangaba.spawn("escreva testes") agente_docs = mangaba.spawn("atualize docs") # Resultados em paralelo resultados = await asyncio.gather( agente_pesquisa, agente_codigo, agente_docs )

Inteligência

200+ modelos de IA disponíveis

Eu uso modelos de IA para "pensar". E você pode escolher qual modelo usar — é como trocar o cérebro do assistente!

Provedor O que é Como usar
OpenRouter 200+ modelos em um só lugar (Claude, GPT, Gemini...) mangaba model set openrouter/anthropic/claude-sonnet-4

O OpenRouter tem um plano gratuito e acesso a mais de 200 modelos. É a melhor opção para começar. Depois você pode trocar para qualquer outro provedor.

Automações

Deixe eu trabalhar sozinho

Você pode me pedir para fazer coisas automaticamente — sem você precisar estar lá. É como ter um funcionário 24 horas.

⏰ Agendador de Tarefas

Configure tarefas para rodar em horários específicos. Funciona como um despertador, mas em vez de tocar, eu executo uma tarefa para você.

# "Todo dia às 9h, me manda um resumo dos emails" mangaba cron add "0 9 * * *" "resumo dos emails" # "Toda sexta às 18h, faz backup dos meus projetos" mangaba cron add "0 18 * * 5" "backup dos projetos" # "A cada hora, verifica se o site está no ar" mangaba cron add "0 * * * *" "verifica site"

🤖 Assistentes Paralelos

Precisa de várias coisas ao mesmo tempo? Eu crio "mini-eus" para trabalhar em paralelo. É como ter uma equipe de assistentes.

# "Pesquise sobre X, escreva testes para Y, # e atualize a documentação de Z — tudo ao mesmo tempo" # Eu crio 3 assistentes trabalhando juntos: # 🤖 Assistente 1: Pesquisa sobre X # 🤖 Assistente 2: Escreve testes para Y # 🤖 Assistente 3: Atualiza documentação

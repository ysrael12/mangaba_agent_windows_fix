Núcleo

AIAgent — O Coração do Sistema

O loop principal de conversação com orquestração de ferramentas.

Componente Principal

Loop de Conversação

O AIAger gerencia o ciclo completo de interação com o LLM.

  • max_iterations: Limite de turnos (padrão: 90)
  • iteration_budget: Controle de custos por sessão
  • interrupt: Pausa/cancelamento em tempo real
  • grace_call: Uma última chance após atingir limite
Gerenciamento

Estado e Sessões

Persistência de dados com SQLite e busca full-text.

  • SessionDB: Armazenamento de sessões
  • FTS5: Busca full-text em conversas
  • Compressão: Otimização de contexto
  • Cache: Prompt caching para economia

Ferramentas

Registry + Toolsets

Sistema de descoberta automática com organização por plataforma.

📋 Registry

Descoberta automática de ferramentas via importação de módulos.

  • Auto-discovery: qualquer tools/*.py com registry.register()
  • Schema collection automática
  • Dispatch e error wrapping
  • Verificação de disponibilidade

📦 Toolsets

Ferramentas organizadas em conjuntos por plataforma.

  • messaging — para Telegram, Discord, Slack
  • terminal — execução de código
  • web — navegação e busca
  • memory — gerenciamento de memória

Plugins

Sistema Extensível

Expanda o agente com novos provedores, memória e capacidades.

Provedores de Memória

Memory Providers

Backends de memória persistentes para o agente.

  • Honcho — modelagem de usuário
  • Mem0 — memória semântica
  • Supermemory — recall inteligente
  • Personalizado via ABC
Provedores de LLM

Model Providers

Adaptadores para cada provedor de modelos.

  • OpenRouter, OpenAI, Anthropic
  • HuggingFace, NVIDIA, Ollama
  • Qualquer API OpenAI-compatível
  • Override via user plugins
Engines de Contexto

Context Engines

Gerenciamento inteligente de contexto.

  • Compressão de conversas
  • Sumarização automática
  • Retrieval Augmented Generation
  • Cache de prompts

🔌 Como Criar um Plugin

# ~/.mangaba/plugins/meu-plugin/plugin.yaml name: meu-plugin kind: model-provider version: 1.0.0 description: Meu provedor personalizado # ~/.mangaba/plugins/meu-plugin/__init__.py from providers import register_provider from providers.base import ProviderProfile profile = ProviderProfile( name="meu-provider", env_vars=("MEU_API_KEY",), base_url="https://api.meuprovedor.com/v1", ) register_provider(profile)
Segurança

Isolamento e Controle

Cada execução é monitorada, isolada e auditada.

🔒 Sandbox

Execução isolada com Docker ou processos separados.

  • Containers isolados
  • Timeout configurável
  • Limites de recursos
  • Cleanup automático

🛡️ Permissões

Controle granular de acesso a ferramentas e ações.

  • Aprovação para ações destrutivas
  • Whitelist de ferramentas
  • Rate limiting por usuário
  • Audit log completo

🔑 Credenciais

Gerenciamento seguro de tokens e chaves de API.

  • Armazenamento em .env
  • Nunca em código ou logs
  • Pool de credenciais
  • Refresh automático

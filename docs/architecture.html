O cérebro: AIAgent

É ele que gerencia toda a conversa. Pensa, decide e executa. É como o "eu" dentro de mim.

O que ele faz

Gerencia a conversa

Quando você me manda uma mensagem, o AIAgent é quem recebe, analisa e decide o que fazer. Ele controla tudo que acontece na nossa conversa.

  • Recebe sua mensagem
  • Analisa o que você quer
  • Decide quais ferramentas usar
  • Executa as ações necessárias
  • Verifica se ficou bom
  • Entrega o resultado
Controle

Segurança e Limites

O AIAgent não é louco — ele tem regras para não fazer besteira. É como um motorista que obedece o código de trânsito.

  • Max 90 rodadas: Não fica em loop infinito
  • Controle de custos: Não gasta mais do que deve
  • Aprovação: Pede antes de fazer coisas perigosas
  • Interrupção: Você pode pausar a qualquer momento

Ferramentas

As mãos: 40+ ferramentas

Cada ferramenta é como um "dedo" meu. Juntas, elas me permitem fazer praticamente qualquer coisa no computador.

📋 Como elas funcionam

Cada ferramenta tem um "cardápio" que diz o que ela faz e como usar. O AIAgent lê esse cardápio e decide qual usar.

# Exemplo: cardápio da ferramenta "ler_arquivo" { "nome": "ler_arquivo", "descrição": "Lê o conteúdo de um arquivo", "como_usar": { "caminho": "caminho do arquivo" } }

📦 Organização

As ferramentas são organizadas em grupos (toolsets). Cada grupo tem ferramentas para uma finalidade diferente.

  • messaging: Enviar mensagens (Telegram, Discord...)
  • terminal: Rodar código (Python, Bash...)
  • web: Acessar a internet (buscar, ler páginas)
  • file: Gerenciar arquivos (ler, escrever, mover)
  • memory: Gerenciar memória (buscar, salvar)
  • todo: Gerenciar tarefas (criar, listar, completar)

📁 Estrutura de Pastas

É assim que os arquivos estão organizados no meu "corpo":

Gateway

A boca: comunicação multi-plataforma

É o componente que me permite conversar com você por qualquer lugar. É como um tradutor que fala todas as "línguas" das plataformas.

Como lembro da conversa

Segurança

Proteção e Controle

Eu não sou um agente selvagem. Tenho regras, limites e proteções para não fazer besteira.

🔒 Sanduíche (Sandbox)

Quando eu rodo código, é numa "jaula" isolada. É como um aquário — eu nadó, mas não fujo.

  • Isolamento: Não afeto o sistema
  • Tempo limite: Não fico preso para sempre
  • Recursos: Não uso mais que devia
  • Limpeza: Tudo é apagado depois

🛡️ Permissões

Nem tudo eu posso fazer. Algumas coisas preciso de permissão.

  • Aprovação: Pergunto antes de deletar
  • Whitelist: Só uso ferramentas autorizadas
  • Limite: Não faço mais de X por hora
  • Log: Tudo fica registrado

🔑 Credenciais

Suas senhas e chaves ficam seguras. Nunca ficam expostas.

  • Arquivo .env: Guardado com segurança
  • Nunca em código: Não escrevo senhas
  • Pool: Uso a chave certa na hora certa
  • Refresh: Atualizo chaves expiradas
